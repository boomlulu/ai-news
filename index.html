据报道：苹果今夜 WWDC 重写 Siri，改投谷歌 Gemini

来源：彭博社 · MacRumors · 9to5Mac2026-06-08（北京时间 6/9 凌晨 keynote）可信度：中高
Lead1 / 3

据多家媒体报道，苹果将在北京时间今夜开幕的 WWDC 2026 上发布重写版 Siri，云端能力改由谷歌定制版 Gemini 驱动。

该定制 Gemini 约 1.2 万亿参数，苹果年付谷歌约 10 亿美元，模型跑在苹果「私有云计算」内、查询不被用于训练；同时推出独立 Siri App 与 Extensions，让用户在 ChatGPT / Gemini / Claude 间切换，Gemini 为默认。

IMPACT 9.3
Analysis2 / 3

看点：全球最大消费电子公司把核心 AI 大脑外包给最大竞争对手之一，等于公开承认自研大模型掉队——这是 AI 竞争格局的标志性转向，也是库克任内的关键一役。

  • 十亿级 iPhone 用户的语音助手底座或首次由谷歌模型驱动，谷歌借此拿下移动端最大入口。
  • Extensions 若开放第三方模型，Anthropic Claude、OpenAI 将争夺苹果设备上的默认与可选位。
  • 苹果「自研 AI」叙事受冲击，资本市场与开发者将重新评估其 AI 路线与议价能力。

两党草案拟「三年冻结」各州 AI 立法，引爆反对

来源：Roll Call · FedScoop · Nextgov · 众议院官网2026-06-04可信度：高
Lead1 / 3

众议员 Obernolte（共和党）与 Trahan（民主党）6 月 4 日联合公布 269 页《伟大的美国 AI 法案》讨论稿，核心是对各州「监管 AI 模型开发」的法律设三年优先豁免。

草案以强制透明度与第三方审计换取冻结，且仅冻结三年（非此前主张的十年），并明确不覆盖针对 AI「使用 / 部署」的州法；但发布数小时内即遭工会、消费者组织乃至众议院民主党委员会近乎一致反对。

IMPACT 9.0
Analysis2 / 3

看点：「联邦统一 vs 各州自治」之争首次落到国会实质性立法文本——叠加特朗普去年 12 月签署的预占式行政令与 DOJ 诉讼小组，AI 监管主导权之争进入摊牌阶段。

  • 若通过，科罗拉多《AI 消费者保护法》（6/30 生效）等州法的「开发」条款或被冻结，企业合规版图将重画。
  • AI 公司或获三年监管确定性预期，但「换审计换冻结」能否在两党两院同时过关存疑。
  • 劳工、儿童与消费者保护团体已组织反对，立法博弈与诉讼或贯穿全年。

英伟达开源 Nemotron 3 Ultra：美国最强开放权重，仍逊中国

来源：Artificial Analysis · 英伟达 · HuggingFace2026-06-04（Computex 6/1 公布）可信度：高
Lead1 / 3

英伟达 6 月 4 日放出 Nemotron 3 Ultra 权重：约 5500 亿参数的开放权重 MoE 推理模型，智能指数 48，为美国开放权重模型最高分。

模型连同训练数据与配方一并发布，许可允许商用，支持 100 万 token 上下文；预发布端点实测 300+ tokens/秒。但全球第一仍是中国 Kimi K2.6（54 分），6 分差距被评测方称「有意义」。

IMPACT 8.1
Analysis2 / 3

看点：英伟达从「卖铲子」延伸到「给最强开源模型」，重塑美国开源叙事；但「美国最强开放权重仍落后中国头部」这一结论，直指开源主导权之争的真实位置。

  • 美国企业获得可商用、可私有部署的高速开放权重选项，降低对闭源 API 的依赖。
  • 英伟达同时掌握算力与开源模型，进一步绑定其 NIM / CUDA 生态。
  • 中美开源模型差距被量化呈现，将牵动 Qwen、DeepSeek、Kimi 等后续迭代节奏。

GitHub Copilot 改按 token 计费，开发者预警账单暴涨

来源：TechCrunch · Visual Studio Magazine · Dataconomy2026-06-01 生效可信度：中高
Lead1 / 3

GitHub Copilot 自 6 月 1 日起由「按请求的固定订阅」改为「按 token 用量」计费，部分开发者警告成本或暴涨 10–50 倍。

有用户称原本月付约 29 美元、新方案估算逼近 750 美元；一名 39 美元 Pro+ 用户称两小时用掉约 8% 月度额度。微软将其与 AI 算力成本上升挂钩，部分开发者已转向 OpenAI Codex、Claude、DeepSeek V4 Pro 或本地开源模型。

IMPACT 7.8
Analysis2 / 3

看点：AI 编程工具「订阅制 → 用量计费」转向的首个标志性事件——AI 真实推理成本正被转嫁给终端开发者，可能重塑整个编码助手市场的价格与忠诚度。

  • 重度 agentic 用户成本结构剧变，需重新评估预算与工具选型。
  • 竞品（Codex、Claude Code、Cursor、本地开源模型）迎来抢量窗口。
  • 用量计费若成范式，将倒逼厂商优化推理成本，也加剧用户对「账单不可预测」的焦虑。
