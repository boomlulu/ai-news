据报道：苹果 WWDC 重写 Siri，改用谷歌 Gemini 大模型

来源：MacRumors · 彭博社（据报道）2026-06-08可信度：中高
Lead1 / 3

据多家媒体报道，苹果将在今天（6/8）开幕的 WWDC 上发布重写版 Siri，云端能力改由谷歌定制版 Gemini（约 1.2 万亿参数）驱动，年付谷歌约 10 亿美元。

据报道还将推出 Extensions 机制，让用户自选 ChatGPT / Gemini / Claude 驱动 Apple Intelligence；Siri 变独立 App，带系统级"问一问"手势与聊天界面。一切以 keynote 正式公布为准。

IMPACT 9.2
Analysis2 / 3

若属实，意味着苹果承认自研 AI 暂时落后，把手机最核心的入口交给对手谷歌——AI 竞争格局的标志性转折。

  • 谷歌 Gemini 借苹果拿到数亿台设备的超级入口。
  • 用户可在系统级自由切换 ChatGPT/Gemini/Claude，模型变"可插拔"。
  • 苹果自研大模型路线受质疑，端云协同策略生变。

Anthropic 发布 Claude Opus 4.8，默认 100 万 token 上下文

来源：Anthropic 官方 · Releasebot2026-06-08可信度：高
Lead1 / 3

Anthropic 发布最新旗舰 Claude Opus 4.8，编程、agent、推理较 Opus 4.7 全面增强，复杂编程任务默认 100 万 token 上下文。

受管 Agent 可在你自己的沙箱内运行、接私有 MCP 服务器；旧版 Opus 4.1 定于 8/5 在 API 退役。

IMPACT 8.3
Analysis2 / 3

编程/agent 是当前最被验证的 AI 商业方向，旗舰每次提速都直接抬高开发者生产力上限。

  • 超长默认上下文降低大型代码库 agent 的工程门槛。
  • 受管 Agent + 私有 MCP 让企业更敢把 agent 接进内网。
  • 旗舰迭代加快，挤压闭源竞品的差异化空间。
Sources3 / 3
主要来源Anthropic 官方 · Releasebot
可信度高

ChatGPT 上线 "Dreaming" V3 记忆系统，容量翻倍

来源：OpenAI 官方 · Releasebot2026-06-04可信度：高
Lead1 / 3

OpenAI 自 2024 年以来最大的记忆升级——Dreaming V3 架构 6/4 起向美国 Plus / Pro 用户推送。

主打解决记忆"过时/出错/难扩展"，做到自动更新、个性化更准、Plus/Pro 记忆容量翻倍。

IMPACT 7.8
Analysis2 / 3

记忆是把 ChatGPT 从"问答工具"变成"长期助理"的关键一步，直接影响数亿人日常使用的黏性与体验。

  • 上下文更新更及时，长期个性化体验提升。
  • 黏性增强，进一步巩固 ChatGPT 的消费级入口地位。
  • 长期记忆带来的隐私与数据边界更受关注。
