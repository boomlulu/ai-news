01 · 商业·并购 · 影响指数 8.7 · 可信度 中高
标题：OpenAI 收购云平台 Ona，Codex 迈向「关机也能跑」
来源：OpenAI 官方博客 · 彭博社 · CNBC
发布时间：2026年6月11日
发生了什么：OpenAI 于 6/11 宣布收购德国云平台 Ona（前身 Gitpod），把其安全云端执行与编排技术并入 Codex。Ona 提供带访问控制、审计日志的预置云环境；并购后 Codex 可承接耗时数小时乃至数天的持久任务，无需绑定单台设备或保持会话在线。OpenAI 称每周已有逾 500 万人使用 Codex，较年初增长 400%；交易金额未披露，仍待惯例交割条件。
为什么重要：编程 Agent 的竞争从「答得准」转向「跑得久」。谁能让 Agent 脱离本机、在云端长时间自治执行，谁就握住企业级落地入口。OpenAI 用一笔收购补齐 Codex 最缺的云端编排底座，直接对标 Anthropic 的 Claude Code。
待验证：交易金额、最终交割时间与团队整合细节未公布；「关机也能跑」的稳定性与安全边界待实测。
建议动作：重度用 Codex/Claude Code 的团队关注云端长任务能力上线节奏，预留迁移评估。

待验证：峰会尚未召开，是否产出具体 AI 条款或联合声明待 6/17 后确认；出席名单仍可能变动。
建议动作：关注 6/15–17 峰会公报中 AI 相关措辞，判断合规与政策风向。

03 · 产品·平台 · 影响指数 7.6 · 可信度 高
标题：Anthropic 计费分离 6/15 生效，Claude Code 开发者倒计时
来源：Anthropic 官方 · Codersera · DigitalApplied
发布时间：2026年6月15日生效（5/14 公告）
发生了什么：Anthropic 自 6/15 起把 Claude Code、Agent SDK、claude -p、GitHub Actions 等程序化用量，从订阅共享池迁出，转入独立计费的「Agent SDK 信用池」。新池按 API 标准价计费、不滚存：Pro 20 美元、Max 5x 100 美元、Max 20x 200 美元；额度用尽后自动化请求即停（除非手动开启溢出计费）。交互式聊天用量不受影响。
为什么重要：订阅制「包月畅跑 Agent」的时代被官方亲手关闭。头部厂商首次正面承认「Agent 烧的算力远超聊天，平价订阅补不起」，所有靠 Claude 订阅跑 CI 与自动化的团队两天内必须重算成本。
待验证：各订阅档实际信用额度与溢出价格以官方结算页为准；迁移对现有 CI 流水线的中断范围待实测。
建议动作：6/15 前盘点所有用 Claude 订阅跑的自动化任务，决定是否开启溢出计费或转 API。
