每日 AI 新闻，2026 年 6 月 13 日，周六。今天没有惊天动地的大模型发布，主线更像一条暗线：Agent 时代的规则，正被几家头部公司同时重写——有人抢入口，有人定规矩，有人开始收以前免费送出去的红利。重点看三件事——OpenAI 收购了德国云平台 Ona，给编程工具 Codex 补云端底座；三家最大的 AI 公司 CEO，要头一回一起去 G7 峰会；还有 Anthropic 的一项计费调整，后天生效，跟很多开发者直接相关。

先说第一件。OpenAI 在六月十一号宣布，收购德国云平台 Ona，它的前身就是不少开发者熟悉的 Gitpod。OpenAI 要把它那套安全的云端执行和编排技术，并进自己的编程工具 Codex。说白了，以后 Codex 能接那种要跑好几个小时、甚至好几天的长任务，不用一直绑在你这台电脑上，会话也不用开着，你电脑关了它也能在云端接着跑。OpenAI 说，现在每周已有超过五百万人在用 Codex，比年初涨了大约四倍。关键在于，编程 Agent 的比拼，正从「答得准不准」转向「能不能自己跑得久」，谁能让 Agent 离开本机、在云端长时间干活，谁就握住企业落地的入口，这一步明显是冲着 Anthropic 的 Claude Code 去的。不过要提醒，这笔交易金额没公布，交割还要走常规流程，稳不稳、安全边界在哪，得等上线后看。

第二件，是一件信号意义很强的事。据彭博社等媒体报道，OpenAI 的奥尔特曼、Anthropic 的阿莫迪、谷歌 DeepMind 的哈萨比斯，都将出席六月十五到十七号、在法国小城埃维昂举办的 G7 峰会。这是三家最前沿的 AI 公司掌门人，头一回一起出现在 G7 这种场合，其中奥尔特曼是应法国总统马克龙私人邀请，第一次参加。Mistral、Meta、Salesforce 这些公司的高管也在名单上。据报道，周三一场工作午餐会专门谈 AI 基建、网络和监管，青少年安全、还有网络和生物领域的前沿风险是核心话题。值得看，是因为当 AI 的估值和算力故事已经冲到万亿级别，监管和产业第一次在 G7 这种最高政治场合正面坐到一张桌子上，这顿闭门午餐谈出的措辞，很可能影响未来一年西方对前沿模型怎么管。当然峰会还没开，最后有没有具体条款或联合声明，要等十七号之后才知道。

第三件，跟开发者钱包直接相关。Anthropic 从六月十五号、也就是后天起，把 Claude Code、Agent SDK，还有那些自动化、跟 GitHub 流水线相关的程序化用量，从原来的订阅共享额度里拿出来，单独放进一个独立计费的信用池。新池按 API 标准价收费，而且用不完不结转，额度一旦用光，自动化请求就会停下，除非你手动打开溢出计费；日常网页聊天的用量不受影响。关键在于，订阅制「包月就能让 Agent 随便跑」的日子，被官方亲手画上句号，这等于头部厂商第一次正面承认，Agent 烧的算力远比聊天多，平价订阅补不起。所有靠 Claude 订阅跑持续集成、跑自动化的团队，这两天得赶紧把成本重新算一遍。

再说几条快读。第一，DeepSeek 的 V4.1，据传可能在六月内做一次小版本更新，重点之一是补上现在还缺的看图能力，时间没坐实。第二，谷歌承诺「六月内」交付的 Gemini 旗舰，主打两百万超长上下文加深度思考，目前还在限量预览，正式开放日期值得盯。第三，接着 G7 那条看，峰会十五到十七号开，重点看 AI 基建和安全会不会写进 G7 的联合表述。

明天继续盯三件事：G7 峰会成果、DeepSeek 版本更新，还有谷歌旗舰什么时候正式开放。以上就是今天的 AI 新闻。
