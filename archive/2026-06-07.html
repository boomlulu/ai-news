OpenAI 模型独立推翻 80 年数学猜想

来源：OpenAI 官方 · Scientific American2026.06.07可信度：高
Lead1 / 3

OpenAI 通用推理模型独立找到 Erdős 1946 年「单位距离猜想」的反例，把上界提升到约 n^1.014。

它用代数数论工具构造出一整族比方格更优的点阵；人类研究这题近 80 年，一直以为方格最优。普林斯顿 Will Sawin 后续精化结果。

IMPACT 9.8
Analysis2 / 3

这是 AI 首次「自主」解决数学子领域核心开放问题，而非当算题助手。Tim Gowers 称之为「AI 数学的里程碑」——AI 开始能在前沿科研里原创。

  • 科研范式可能改变：AI 从检索/计算工具，升级为能提出新结构的「发现者」。
  • 跨学科迁移：它借用了看似无关的数论分支来解几何题。
  • 正面回应了「AI 只会复述已知」的长期质疑。

主要来源OpenAI 官方 · Scientific American
可信度高
待验证该能力能否复现到其它难题、是否依赖大量人工引导，仍待观察。

MiniMax M3 开源：逼近顶级闭源，成本仅 5%–10%

来源：devFlokers 开源月报 · llm-stats2026.06.07可信度：中高
Lead1 / 3

国产 MiniMax 开放权重模型 M3：SWE-Bench Pro 拿到 59.0%，超过 GPT-5.5 与 Gemini 3.1 Pro 等闭源 API。

支持 100 万 token 上下文 + 原生多模态「电脑操作」；靠 MSA 稀疏注意力，把长文本算力砍到上一代约 1/20。

IMPACT 9.3
Analysis2 / 3

完全符合「又强又便宜」的爆点标准：先得很强，其次部署成本极低。普通团队也用得起接近顶级的能力，重塑整个成本结构。

  • 推理成本骤降，中小团队可负担顶级能力。
  • 长上下文 + 多模态 agent 的门槛大幅降低。
  • 开放权重加速生态，但训练代码未开源——严格说是「开放权重」而非「完全开源」。
Sources3 / 3
主要来源devFlokers 开源月报 · llm-stats
可信度中高
待验证真实业务下的稳定性、安全对齐、与基准成绩的差距，待第三方验证。
建议动作评估用 M3 替换部分闭源调用以降本；实测其 agent / 电脑操作的真实稳定性。
