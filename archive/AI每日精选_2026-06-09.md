### 1. OpenAI 称「聊天已死」，ChatGPT 将重写为 Agent 超级应用

- **分类**：产品·商业 ｜ **影响指数**：9.1 ｜ **可信度**：中高
- **发生了什么**：OpenAI 内部称「聊天已死（Chat is dead）」，启动 ChatGPT 上线以来最大改版，将其重写为以 Agent 为核心的「超级应用」。据金融时报，新版拟合并 ChatGPT、Codex 编程平台与 Atlas 浏览器三条产品线，接入 Canva、Booking.com 等合作方应用；公司已内部重组，把相关团队统一交由新任首席产品官 Thibault Sottiaux。企业用户已占其营收 **40%**，目标年底升至 **50%**，改版正值其筹备年内 IPO。
- **为什么重要**：头部实验室第一次公开宣判「对话框」范式终结——竞争主轴从「谁答得好」转向「谁能替你把事办完」，也是 OpenAI 给 IPO 投资人「ChatGPT 不止是聊天玩具」的答案。
- **待验证**：新版界面与「超级应用」尚未上线（官方称未来数周推出），合作方清单与 Atlas 整合细节待落地。
- **来源**：金融时报 · TechCrunch · The Decoder

### 2. ChatGPT 上线「锁定模式」，断网防提示注入窃密

- **分类**：安全 ｜ **影响指数**：7.9 ｜ **可信度**：高
- **发生了什么**：OpenAI 为 ChatGPT 推出可选的「锁定模式（Lockdown Mode）」，通过限制对外网络请求来降低提示注入导致的数据外泄风险。开启后将停用实时联网浏览（仅访问缓存）、联网取图、深度研究与 Agent 模式，且不能下载文件做数据分析；在「设置 > 安全」中开启。OpenAI 明确警告：即便开启，缓存内容或上传文件中的注入仍可能影响回答。
- **为什么重要**：当 Agent 被允许联网与操作工具，提示注入就从「理论风险」变成「会偷数据的真攻击」——头部厂商把「主动断网」做成产品开关，等于承认现有防护挡不住，先给高敏用户兜底。
- **待验证**：实际防护强度与对正常工作流的影响，需安全团队实测评估。
- **来源**：OpenAI · TechCrunch · The Decoder

### 3. Perplexity 推「搜索即代码」，让模型自己写检索管线

- **分类**：研究·产品 ｜ **影响指数**：7.7 ｜ **可信度**：高
- **发生了什么**：Perplexity 发布「Search as Code」，让 AI Agent 自行生成 Python 脚本来调度搜索，取代固定的搜索 API。架构由推理模型作「控制面」、安全沙箱执行、模块化 Agentic Search SDK 三部分组成。官方称在一项复杂 CVE 研究任务上，token 用量较标准管线下降 **85%**。该能力已并入 Agent API，并成为其 Agent 平台 Computer 的默认搜索方式。
- **为什么重要**：检索从「调用固定 API」升级为「模型现写现跑代码」，是 Agent 把外部工具内化为可编程能力的方向标，也直指 RAG/搜索接口的下一代形态。
- **待验证**：85% 省 token 为官方单任务数据，跨场景表现与沙箱安全性需第三方验证。
- **来源**：Perplexity Research · The Decoder · Winbuzzer

### 4. Anthropic 挖走 OpenAI「第二号」自研芯片工程师

- **分类**：商业·公司动态 ｜ **影响指数**：7.5 ｜ **可信度**：中高
- **发生了什么**：自称 OpenAI 自研芯片项目「第二位硬件员工」的 Clive Chan 离职加入 Anthropic，正值两家公司同步冲刺 IPO。Chan 此前在特斯拉 Autopilot 做 ML 训练定制芯片，2024 年 1 月加入 OpenAI，参与从零搭建定制芯片并涉及 OpenAI–博通合作。Anthropic 据报道正考虑自研 AI 芯片（截至 4 月仍处早期、尚无专职团队）。
- **为什么重要**：IPO 前争抢稀缺的「自研芯片」人才，是头部实验室向公开市场证明「正在降低对英伟达依赖、构建长期算力护城河」的信号——人才流向，就是战略下注方向。
- **待验证**：Chan 在 Anthropic 的具体职责、Anthropic 是否成立自研芯片专职团队，均未官方确认。
- **来源**：The Decoder · Benzinga · Dealroom
