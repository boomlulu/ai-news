OpenAI 收购云平台 Ona，Codex 迈向「关机也能跑」

来源：OpenAI 官方博客 · 彭博社 · CNBC2026年6月11日可信度：中高
Lead1 / 3

OpenAI 于 6/11 宣布收购德国云平台 Ona（前身 Gitpod），把其安全云端执行与编排技术并入 Codex。

Ona 提供带访问控制、审计日志的预置云环境；并购后 Codex 可承接耗时数小时乃至数天的持久任务，无需绑定单台设备或保持会话在线。OpenAI 称每周已有逾 500 万人使用 Codex，较年初增长 400%；交易金额未披露，仍待惯例交割条件。

IMPACT 8.7
Analysis2 / 3

看点：编程 Agent 的竞争从「答得准」转向「跑得久」——谁能让 Agent 脱离本机、在云端长时间自治执行，谁就握住企业级落地入口。OpenAI 用一笔收购补齐 Codex 最缺的云端编排底座，直接对标 Anthropic 的 Claude Code。

  • 开发者可把数小时/数天的长任务丢给 Codex 云端跑，本机关机不影响。
  • Gitpod/Ona 既有云开发环境用户面临产品归属与路线调整。
  • 编程 Agent 主战场从「单次补全」转入「持久自治执行」，抬高对手门槛。

三大 AI CEO 首次同赴 G7，奥尔特曼初登峰会

来源：彭博社 · 法新社 · The Next Web2026年6月12日可信度：中高
Lead1 / 3

据彭博社等，OpenAI 奥尔特曼、Anthropic 阿莫迪、谷歌 DeepMind 哈萨比斯将出席 6/15–17 在法国 Évian 举行的 G7 峰会。

这是三大前沿 AI 公司 CEO 首次同台 G7；奥尔特曼系应马克龙私人邀请首度与会。Mistral、Cohere、Meta、Salesforce 等高管亦在列。周三工作午餐将聚焦 AI 基建、网络与监管；青少年安全、网络与生物领域前沿风险是核心议题。

IMPACT 8.0
Analysis2 / 3

看点：当 AI 估值与算力叙事冲到万亿级，监管与产业第一次在 G7 这种最高政治场合正面坐到一张桌上——这场闭门午餐谈出的措辞，可能定调未来一年西方对前沿模型的治理口径。

  • 前沿模型安全、青少年保护或进入 G7 联合表述，影响后续立法走向。
  • 三家竞争对手罕见同场，行业自律与游说格局浮出水面。
  • 欧洲（马克龙、Mistral）借主场强化在 AI 治理中的话语权。

Anthropic 计费分离 6/15 生效，Claude Code 开发者倒计时

来源：Anthropic 官方 · Codersera · DigitalApplied2026年6月15日生效（5/14 公告）可信度：高
Lead1 / 3

Anthropic 自 6/15 起把 Claude Code、Agent SDK、claude -p、GitHub Actions 等程序化用量，从订阅共享池迁出，转入独立计费的「Agent SDK 信用池」。

新池按 API 标准价计费、不滚存：Pro $20、Max 5x $100、Max 20x $200；额度用尽后自动化请求即停（除非手动开启溢出计费）。交互式聊天用量不受影响。此举意在缓解 Agent 高算力消耗带来的订阅可持续性压力。

IMPACT 7.6
Analysis2 / 3

看点：订阅制「包月畅跑 Agent」的时代被官方亲手关闭——这是头部厂商首次正面承认「Agent 烧的算力远超聊天，平价订阅补不起」，所有靠 Claude 订阅跑 CI/自动化的团队两天内必须重算成本。

  • 用订阅额度跑 CI/CD、批量 Agent 的团队，6/15 起可能突然「断流」。
  • 各家或跟进「交互与程序化分开计费」，Agent 成本透明化。
  • 独立开发者需在溢出计费与 API 直连间重新选型。
