字节跳动转购国产 AI 芯片，绕开英伟达

来源：路透社 · The Next Web · Investing.com2026年6月15日可信度：中高
Lead1 / 3

据路透社独家，字节跳动正与上海天数智芯（Iluvatar CoreX）洽谈采购至少 5 万颗 AI 芯片，主要用于推理、支撑其豆包大模型；同时评估改用百度昆仑芯。

若达成，天数智芯将成为字节继华为、寒武纪之后的第三家主力国产 GPU 供应商；腾讯已是昆仑芯客户。谈判仍在进行、条款未定。背景是美国对华先进芯片出口管制持续收紧，中国头部 AI 公司正加速「去英伟达」、转向本土算力。

IMPACT 8.8
Analysis2 / 3

这不是一笔普通采购，而是全球最大 AI 应用公司之一用真实订单为国产芯片「背书」——当美国卡住高端 GPU，中国需求端正主动把算力供应链搬回国内，全球 AI 硬件的「脱钩」从政策层落到了订单层。

  • 国产替代提速：字节级别的推理订单能直接养活天数智芯、昆仑芯等本土厂商，缩小与英伟达的代差窗口。
  • 供应链分叉：中国推理算力加速形成「华为＋寒武纪＋天数智芯/昆仑芯」本土矩阵，全球硬件市场进一步双轨化。
  • 英伟达承压：中国大客户用脚投票转向国产，叠加出口管制，其在华推理市场份额面临结构性流失。

亚马逊举报「越狱」，白宫叫停 Anthropic 最强模型

来源：Axios · CNBC · Anthropic 官方2026年6月15日可信度：高
Lead1 / 3

围绕白宫叫停 Anthropic 旗舰模型 Mythos 5 与 Fable 5 的风波，新进展浮出：触发点是亚马逊——它向白宫递交报告，称研究员「越狱」Mythos、套出本应受限的网络攻击信息。

据 Axios、CNBC、Fortune，商务部长卢特尼克致信 CEO 阿莫迪，对两款模型实施出口管制、禁止任何外国人访问，Anthropic 被迫对所有客户停用。Semafor 称白宫核心顾虑是中国可能借此获取能力。截至 6/16，Anthropic 高级技术团队已赴华盛顿磋商，双方均称想尽快解决，但禁令尚未解除。

IMPACT 8.5
Analysis2 / 3

看点在于「谁按下了暂停键」——亚马逊是 Anthropic 最大投资方之一，却亲手把自家被投企业的旗舰举报到白宫；而政府动用出口管制的真正动机直指防范中国。这让「前沿模型一上线就可能被竞争对手＋监管联手召回」从假设变成现实。

  • 投资即对手：大厂既投又用还能举报被投模型，前沿实验室的「盟友」关系被重新定义，安全披露可成竞争武器。
  • 召回成风险：模型上线后仍可被一纸指令叫停，所有前沿厂商的部署都多了一重「随时下架」的不确定性。
  • 对华是主线：管制以「防中国获取」为由，意味着能力越强的模型，海外（尤其涉华）可用性越受限。

月之暗面开源 Kimi K2.7 Code，却回避公开榜单

来源：Moonshot 官方 · MarkTechPost · DevOps.com2026年6月12日可信度：中高
Lead1 / 3

月之暗面 6/12 在 Hugging Face 开源 Kimi K2.7 Code，主打智能体编码：保留万亿参数 MoE 架构，长程软件工程能力增强，推理 token 用量较 K2.6 降约 30%。

官方自报较 K2.6 提升明显：Kimi Code Bench v2 +21.8%、Program Bench +11.0%、MLS Bench Lite +31.5%，并新增 HighSpeed 模式；采用「改良版 MIT」许可，可经 Kimi API 与 Kimi Code CLI 调用。但截至发布，全部跑分均来自月之暗面自家基准，SWE-bench、Terminal-Bench 等公开榜单暂无第三方数据。

IMPACT 7.5
Analysis2 / 3

看点不在「又快又省」，而在「只晒自家分、不交公开榜」——上一代 K2.6 曾以 SWE-bench Verified 80.2% 坐上全球最强开源编码模型；这次官方主动回避独立验证，能力是真涨还是「选择性跑分」，成了开发者必须自测的问号。

  • 开源编码仍由中国队领跑节奏：在闭源旗舰扎堆前夜，月之暗面用月级迭代守住开源编码心智。
  • 省 token 直击成本：智能体编码按 token 计费，省量即省钱，对重度 CI/自动化团队有实在吸引力。
  • 信任要打折：缺第三方公榜，「自报提升」需谨慎，选型前务必用自己的任务集实测。
