## 🔥 头条

### 1. OpenAI 收购云平台 Ona，Codex 迈向「关机也能跑」

- **分类**：商业·并购 ｜ **影响指数**：8.7 ｜ **可信度**：中高
- **发生了什么**：OpenAI 于 6/11 宣布收购德国云平台 **Ona**（前身 Gitpod），把其安全云端执行与编排技术并入 **Codex**。Ona 提供带访问控制、审计日志的预置云环境；并购后 Codex 可承接耗时数小时乃至数天的持久任务，无需绑定单台设备或保持会话在线。OpenAI 称每周已有逾 **500 万**人使用 Codex，较年初增长 **400%**；交易金额未披露，仍待惯例交割条件。
- **为什么重要**：编程 Agent 的竞争从「答得准」转向「跑得久」——谁能让 Agent 脱离本机、在云端长时间自治执行，谁就握住企业级落地入口。OpenAI 用一笔收购补齐 Codex 最缺的云端编排底座，直接对标 Anthropic 的 Claude Code。
- **待验证**：交易金额、最终交割时间与团队整合细节未公布；「关机也能跑」的稳定性与安全边界待实测。
- **来源**：OpenAI 官方博客 · 彭博社 · CNBC

### 2. 三大 AI CEO 首次同赴 G7，奥尔特曼初登峰会

- **分类**：政策·治理 ｜ **影响指数**：8.0 ｜ **可信度**：中高
- **发生了什么**：据彭博社等，OpenAI 奥尔特曼、Anthropic 阿莫迪、谷歌 DeepMind 哈萨比斯将出席 **6/15–17** 在法国 **Évian** 举行的 G7 峰会。这是三大前沿 AI 公司 CEO **首次**同台 G7；奥尔特曼系应马克龙私人邀请**首度**与会。Mistral、Cohere、Meta、Salesforce 等高管亦在列。周三工作午餐聚焦 AI 基建、网络与监管；青少年安全、网络与生物领域前沿风险是核心议题。
- **为什么重要**：当 AI 估值与算力叙事冲到万亿级，监管与产业第一次在 G7 这种最高政治场合正面坐到一张桌上——这场闭门午餐谈出的措辞，可能定调未来一年西方对前沿模型的治理口径。
- **待验证**：峰会尚未召开，是否产出具体 AI 条款或联合声明待 6/17 后确认；出席名单仍可能变动。
- **来源**：彭博社 · 法新社 · The Next Web

### 3. Anthropic 计费分离 6/15 生效，Claude Code 开发者倒计时

- **分类**：产品·平台 ｜ **影响指数**：7.6 ｜ **可信度**：高
- **发生了什么**：Anthropic 自 **6/15** 起把 Claude Code、Agent SDK、claude -p、GitHub Actions 等程序化用量，从订阅共享池迁出，转入独立计费的「**Agent SDK 信用池**」。新池按 API 标准价计费、**不滚存**：Pro $20、Max 5x $100、Max 20x $200；额度用尽后自动化请求即停（除非手动开启溢出计费）。交互式聊天用量不受影响。
- **为什么重要**：订阅制「包月畅跑 Agent」的时代被官方亲手关闭——头部厂商首次正面承认「Agent 烧的算力远超聊天，平价订阅补不起」，所有靠 Claude 订阅跑 CI/自动化的团队两天内必须重算成本。
- **待验证**：各订阅档实际信用额度与溢出价格以官方结算页为准；迁移对现有 CI 流水线的中断范围待实测。
- **来源**：Anthropic 官方 · Codersera · DigitalApplied
