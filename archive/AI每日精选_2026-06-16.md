## 🔥 头条

### 1. 字节跳动转购国产 AI 芯片，绕开英伟达

- **分类**：商业·芯片 ｜ **影响指数**：8.8 ｜ **可信度**：中高
- **发生了什么**：据**路透社**独家，字节跳动正与上海**天数智芯（Iluvatar CoreX）**洽谈采购至少 **5 万颗** AI 芯片，主要用于推理、支撑其豆包大模型；同时评估改用**百度昆仑芯**。若达成，天数智芯将成为字节继华为、寒武纪之后的**第三家**主力国产 GPU 供应商（腾讯已是昆仑芯客户）。谈判仍在进行、条款未定，背景是美国对华先进芯片出口管制持续收紧。
- **为什么重要**：这不是一笔普通采购，而是全球最大 AI 应用公司之一用真实订单为国产芯片「背书」——当美国卡住高端 GPU，中国需求端正主动把算力供应链搬回国内，全球 AI 硬件的「脱钩」从政策层落到了订单层。
- **3 条影响**：① 国产替代提速，字节级别的推理订单直接养活天数智芯、昆仑芯等本土厂商；② 供应链分叉，中国推理算力加速形成「华为＋寒武纪＋天数智芯/昆仑芯」本土矩阵；③ 英伟达承压，中国大客户用脚投票转向国产，叠加管制，在华推理份额面临结构性流失。
- **待验证**：采购数量、金额、是否最终签约均为路透社消息源口径，字节与各芯片厂商未公开确认，交易仍可能生变。
- **建议动作**：关注国产推理芯片产能与生态成熟度；依赖英伟达在华供货的团队预判份额与价格变化，评估多供应商冗余。
- **来源**：路透社 · The Next Web · Investing.com

### 2. 亚马逊举报「越狱」，白宫叫停 Anthropic 最强模型

- **分类**：政策·监管 ｜ **影响指数**：8.5 ｜ **可信度**：高
- **发生了什么**：围绕白宫叫停 Anthropic 旗舰模型 **Mythos 5 与 Fable 5** 的风波，新进展浮出：触发点是**亚马逊**——它向白宫递交报告，称研究员「越狱」Mythos、套出本应受限的网络攻击信息。据 Axios、CNBC、Fortune，商务部长卢特尼克致信 CEO 阿莫迪，对两款模型实施出口管制、禁止任何外国人访问，Anthropic 被迫对所有客户停用。**Semafor** 称白宫核心顾虑是**中国**可能借此获取能力。截至 6/16，Anthropic 高级技术团队已赴华盛顿磋商，双方均称想尽快解决，但禁令**尚未**解除。
- **为什么重要**：看点在于「谁按下了暂停键」——亚马逊是 Anthropic 最大投资方之一，却亲手把自家被投企业的旗舰举报到白宫；而政府动用出口管制的真正动机直指防范中国。这让「前沿模型一上线就可能被竞争对手＋监管联手召回」从假设变成现实。
- **3 条影响**：① 投资即对手，大厂既投又用还能举报被投模型，安全披露可成竞争武器；② 召回成风险，模型上线后仍可被一纸指令叫停，前沿部署多一重不确定性；③ 对华是主线，能力越强的模型，海外（尤其涉华）可用性越受限。
- **待验证**：越狱严重程度双方各执一词；华盛顿磋商结果、禁令是否细化或撤销均未定；亚马逊角色细节以媒体消息源为准。
- **建议动作**：依赖 Mythos/Fable 的产品继续用备份模型；前沿选型纳入「监管召回」预案；涉华业务评估高能力模型的可用性风险。
- **来源**：Axios · CNBC · Anthropic 官方（Fortune、Semafor 跟进）

### 3. 月之暗面开源 Kimi K2.7 Code，却回避公开榜单

- **分类**：模型·开源 ｜ **影响指数**：7.5 ｜ **可信度**：中高
- **发生了什么**：月之暗面 **6/12** 在 Hugging Face 开源 **Kimi K2.7 Code**，主打智能体编码：保留万亿参数 MoE 架构，长程软件工程能力增强，推理 token 用量较 K2.6 降约 **30%**。官方自报较 K2.6 提升明显：Kimi Code Bench v2 **+21.8%**、Program Bench +11.0%、MLS Bench Lite +31.5%，并新增 HighSpeed 模式；采用「改良版 MIT」许可，可经 Kimi API 与 Kimi Code CLI 调用。但截至发布，**全部跑分均来自月之暗面自家基准**，SWE-bench、Terminal-Bench 等公开榜单暂无第三方数据。
- **为什么重要**：看点不在「又快又省」，而在「只晒自家分、不交公开榜」——上一代 K2.6 曾以 SWE-bench Verified 80.2% 坐上全球最强开源编码模型；这次官方主动回避独立验证，能力是真涨还是「选择性跑分」，成了开发者必须自测的问号。
- **3 条影响**：① 开源编码仍由中国队领跑节奏，月之暗面用月级迭代守住开源编码心智；② 省 token 直击成本，智能体编码按 token 计费，省量即省钱；③ 信任要打折，缺第三方公榜，「自报提升」需谨慎。
- **待验证**：+21.8% 等均为自家基准，无 SWE-bench/Terminal-Bench 等独立第三方数据；HighSpeed 模式与「省 30% token」的真实表现待复测。
- **建议动作**：想上开源编码模型的团队，用自有任务集对比 K2.7 Code 与 K2.6/闭源，别仅凭官方跑分决策。
- **来源**：Moonshot 官方 · MarkTechPost · DevOps.com
